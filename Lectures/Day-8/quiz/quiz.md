1. 다음 코드의 출력 결과는 무엇인가?

```python
class A:
    def __init__(self):
        print("A")

class B(A):

    def __init__(self):
        print("B")
        super().__init__()

b = B()
```

1) A
2) B
3) B, A
4) A, B
5) None
